## Description

A free and open-source project to make votes task points with poker style.

## Prerequisites

- npm `npm i -g npm`
## Getting Started

```
npm i
```
This command will install everything needed for the project, based on package.json

## Run
### In Local with Local

```
npm run local
```

### In Production

```
npm run prod
```

## Redis
### For Local

```
npm run init-redis
```
